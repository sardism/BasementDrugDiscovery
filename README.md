BasementDrugDiscovery

One scientist. One laptop. $20/month AI. 20 years of protein expertise.

Can a single person build a drug discovery pipeline using only open source tools, free databases, and a consumer GPU and find something real?

This project screens all FDA approved drugs against validated protein targets in fungal pathogens responsible for killing immunocompromised patients worldwide. Every drug in the library has already cleared human safety trials. If something works computationally, the path to patients is years shorter than starting from scratch.

The pipeline is fully open. Every notebook, every result, every decision is documented in real time on Substack. Anyone can reproduce this on their own laptop.

Current targets: CYP51 from Candida albicans, Aspergillus fumigatus, and human CYP51 as a selectivity counter-screen.

Current top hit: Atogepant, an FDA approved migraine drug, scores stronger than every known antifungal against both fungal targets. Human CYP51 counter-screen in progress.

This is a response to FDA Docket FDA-2026-N-4492, Drug Repurposing for Unmet Medical Needs.
